## 📁 Структура проекта

```
openclaw.ru/
├── src/                    # Frontend (Next.js 14 + App Router)
│   ├── app/               # Страницы и layout
│   ├── components/        # React компоненты
│   │   ├── dashboard-stats.tsx
│   │   ├── active-sessions.tsx
│   │   ├── recent-tasks.tsx
│   │   ├── system-status.tsx
│   │   └── providers.tsx

│   └── index.js          # API endpoints для OpenClaw

└── конфигурационные файлы
```

## 🔧 Технический стек

### Frontend:
- **Next.js 14** с App Router
- **TypeScript** для типобезопасности
- **Tailwind CSS** для стилизации
- **React Query** для управления состоянием
- **Lucide React** для иконок

### Backend:
- **Express.js** для API сервера
- **CORS** для кросс-доменных запросов
- **Интеграция с OpenClaw CLI**

### Инструменты:
- **Git** для контроля версий
- **GitHub CLI (gh)** для автоматизации
- **npm scripts** для управления задачами

### Для работы с Cursor на другом компьютере:
1. **Клонировать репозиторий:**
   ```bash
   git clone https://github.com/rustamkvk/openclaw.ru.git
   cd openclaw.ru
   npm install
   ```

2. **Настроить окружение:**
   - Установить Node.js 18+
   - Настроить доступ к OpenClaw (если нужен)
   - Создать `.env.local` при необходимости

3. **Разработка:**

   - Создавать Pull Requests
