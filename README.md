# OpenClaw Dashboard (ЛК)

Дашборд управления OpenClaw - веб-интерфейс для мониторинга и управления агентами, задачами и памятью.

## 🚀 Функциональность

### Основные модули:
1. **Контрольный центр** - мониторинг всех активных сессий и субагентов
2. **Управление памятью** - просмотр и редактирование MEMORY.md и daily notes
3. **Задачи и конвейеры** - управление задачами 44-ФЗ и другими конвейерами
4. **Интеграции** - статус Telegram, VK, VPN и других подключений
5. **Настройки** - конфигурация моделей, навыков и инструментов

### Технический стек:
- **Frontend:** React + TypeScript + Tailwind CSS
- **Backend:** Node.js + Express (интеграция с OpenClaw API)
- **База данных:** SQLite (для хранения состояния)
- **Реальное время:** WebSocket для live обновлений

## 🛠️ Установка и запуск

```bash
# Клонирование репозитория
git clone https://github.com/rustamkvk/openclaw.ru.git
cd openclaw.ru

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для production
npm run build
```

## 🔌 Интеграция с OpenClaw

Проект интегрируется с OpenClaw через:
1. **OpenClaw API** - для управления агентами и задачами
2. **Файловая система** - для доступа к рабочей области
3. **WebSocket** - для real-time уведомлений
