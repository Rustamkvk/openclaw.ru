## Предлагаемая структура меню:

### 🏠 ГЛАВНАЯ
- **Dashboard** - общая статистика
- **Профиль** - настройки пользователя

### 🤖 АГЕНТЫ И СИСТЕМА
- **Агенты** - список и статус агентов
- **Мониторинг** - метрики системы
- **Логи** - просмотр логов API
- **Логирование** - расширенное логирование

### 📁 44-ФЗ И ЗАДАЧИ
- **44-ФЗ: Обзор** - общая информация
- **44-ФЗ: Задачи** - управление задачами
- **44-ФЗ: Документы** - работа с документами
- **44-ФЗ: Аналитика** - аналитика закупок

### 💬 ВЗАИМОДЕЙСТВИЕ
- **Чат с Атласом** - прямой чат
- **Задачи для Атласа** - постановка задач
- **Результаты** - просмотр выполненных задач

### 🧠 ПАМЯТЬ И НАСТРОЙКИ
- **Память Атласа** - просмотр памяти
- **Настройки системы** - конфигурация
- **Пользователи** (admin only)
- **Резервные копии**

### 🔧 АДМИНИСТРИРОВАНИЕ (admin only)
- **Пользователи** - управление пользователями
- **Настройки API** - конфигурация API
- **Мониторинг** - системный мониторинг
- **Логи** - детальные логи
